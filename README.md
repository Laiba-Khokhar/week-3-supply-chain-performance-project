
# 📦 Supply Chain Analytics Dashboard

## 🎯 Objectives

- Clean and preprocess a real-world supply chain dataset.
- Perform exploratory data analysis (EDA).
- Analyze shipment performance and logistics costs.
- Build an interactive Power BI dashboard.
- Generate business insights and recommendations.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Microsoft Power BI
- Git & GitHub

---

## 📂 Dataset

The dataset contains real-world supply chain information, including:

- Shipment Details
- Vendor Information
- Product Details
- Delivery Dates
- Freight Cost
- Shipment Mode
- Country Information
- Line Item Value

---

# 🐍 Part 1: Data Analysis with Python

## Data Loading

- Loaded the dataset using Pandas.
- Displayed sample records.
- Explored dataset dimensions and column information.

## Data Cleaning

The following preprocessing steps were performed:

- Converted date columns into datetime format.
- Handled missing values.
- Converted numeric columns into appropriate data types.
- Replaced invalid values with nulls where necessary.
- Prepared the dataset for analysis.

---

## 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

### Shipment Analysis

- Most common shipment mode
- Total shipments by country

### Delivery Performance

- Delivery delay calculation
- Average delivery time
- Delayed shipment identification

### Cost Analysis

- Total Freight Cost
- Average Line Item Value
- Total Insurance Cost

### Group-Based Analysis

- Country vs Total Shipments
- Vendor vs Total Freight Cost
- Product Group vs Total Value
- Shipment Mode vs Average Delivery Delay

---

## 📈 Python Visualizations

The following charts were created:

- Bar Chart – Country vs Total Shipments
- Pie Chart – Shipment Mode Distribution
- Line Chart – Delivery Trend Over Time
